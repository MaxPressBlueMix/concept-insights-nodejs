IBM Cloud BP Solution Explorer powered by Watson

IBM Cloud BP Solution Explorer - Technology Demo

IBM Cloud BP Solution Explorer will be a cognitive computing feature of the new IBM Cloud Business Partner Guide powered by Watson. It uses IBM Watson's Concept Insights service to search for IBM Cloud solutions based on relevant concepts.

 

Concept Search

The IBM Watson Concept Insights API in this application has built a body of knowledge around IBM Cloud Marketplace solutions offered by IBM. It is not just using keyword searching.

You can find solutions by querying concepts relevant to a business need even if those concepts are not actually in any database.
